# fast-net-tools

## Usage
```
usage: fastnettool.py [-h] [-t] [-T MAX_TTL] [-c COUNTS] address

positional arguments:
  address               Destination host address

optional arguments:
  -h, --help            show this help message and exit
  -t, --traceroute      Execute traceroute tool for `address`
  -T MAX_TTL, --max_ttl MAX_TTL
                        Max TTL counter
  -c COUNTS, --count COUNTS
                        Number of pings for each router
```
